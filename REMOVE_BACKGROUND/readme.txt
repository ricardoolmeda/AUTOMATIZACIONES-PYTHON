# Como usarlo

Para usarlo solo hay que ejecutar el .py ya sea mediante la consola
o mediante nuestra interfaz de programación.


## Como funciona

Lo que hace el script a la hora de ejecutarlo es buscar los archivos
.jpg y .png que estén en la misma carpeta que el script y seguidamente
crea una carpeta llamada "Fondo Removido" donde guardará automáticamente
las imagenes sin el fondo.

## Versiones usadas
Python 3.9 (debido a la compatibilidad con las librerias)
pip install rembg 
pip install pillow
